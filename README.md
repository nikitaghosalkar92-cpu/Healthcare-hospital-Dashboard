# Healthcare-hospital-Dashboard
Provides a visual summary of overall patient count and hospital count.

Shows patient demographics including gender and blood group distribution.

Displays medical conditions with patient counts for each category.

Tracks patient admissions and discharges across multiple years.

Presents hospital-wise revenue and insurance-wise average billing.

Includes charts for medication usage based on test results.

Built with interactive Excel elements like dropdown filters for easy analysis.

Offers a clean, professional dashboard layout ideal for quick healthcare insights.
